# Zote Precepts

A solemn memorial for a wise grey prince

## Installation

This repository requires git-lfs before cloning

1. Install git-lfs (https://git-lfs.com/)
2. Run `git lfs install`
3. You're OK to clone

Once the repo is cloned, run the project using

```sh
npm install
npm run dev
```
